# PyLock 🔐

A CLI-based security toolkit built with Python.

## Features

* Password generator
* Username generator
* Passphrase generator
* Password strength checker
* Password hashing (bcrypt)

## Installation

```bash
pip install -e .
```

## Usage

```bash
pylock
```
